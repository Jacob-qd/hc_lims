## 迭代计划

### Phase G1: P0 差距补全 (3天)

#### G1.1 条码/二维码生成打印

**文档先行**: 更新 SYSTEM_DESIGN.md §3.2 样品管理 — 添加条码设计

**开发内容**:
- 条码生成组件 (使用 jsbarcode)
- 样品标签打印功能
- 条码扫描输入支持

**验收标准**:
- [ ] 样品登记时自动生成条码
- [ ] 支持批量打印条码标签
- [ ] 条码扫描可快速定位样品
- [ ] 条码格式可配置

---

#### G1.2 COC 监管链

**文档先行**: 更新 SYSTEM_DESIGN.md §3.2 样品管理 — 添加COC设计

**开发内容**:
- COC 交接记录表单
- 样品签收/移交时间线
- 交接人电子签名
- COC 报告导出

**验收标准**:
- [ ] 每次样品交接有记录
- [ ] 交接人签名+时间戳
- [ ] COC 完整可追溯
- [ ] COC 报告可导出

---

#### G1.3 批量操作增强

**文档先行**: 更新 SYSTEM_DESIGN.md — 批量操作规范

**开发内容**:
- 表格行多选 (selection)
- 批量状态变更
- 批量导入 (CSV/Excel)
- 批量导出 (CSV/Excel/PDF)
- 批量删除 (确认弹窗)

**验收标准**:
- [ ] 所有列表页支持多选
- [ ] 批量导入支持CSV/Excel
- [ ] 批量导出支持CSV/Excel/PDF
- [ ] 批量操作有进度提示
