## 三、手动验证路径

### 3.1 预警中心
1. 打开 http://localhost:5173/dashboard
2. 确认顶部出现"⚠️ 预警中心"卡片行
3. 点击"质控违规"卡片 → 跳转到 /quality
4. 点击"校准到期"卡片 → 跳转到 /instruments

### 3.2 分样与留样
1. 打开 http://localhost:5173/samples
2. 底部出现"🧪 分样管理"和"📦 留样管理"两个Tab
3. 分样Tab: 查看子样品Tag (A:200mL→COD+氨氮)
4. 留样Tab: 查看到期天数颜色 (正常=绿, 即将到期=黄, 过期=红)
5. 点击"处置"按钮→弹出确认Modal

### 3.3 报告作废与分发
1. 打开 http://localhost:5173/reports
2. 勾选报告 → 点击"作废重签" → 弹出确认弹窗
3. 勾选报告 → 点击"分发报告" → 弹出确认弹窗

### 3.4 超标复测
1. 打开 http://localhost:5173/tasks/tk2/result
2. 将某个检测指标的判定改为"超标" (如有)
3. 点击"提交复核" → 弹出超标确认Modal → 选择"创建复测"

### 3.5 CAPA闭环
1. 打开 http://localhost:5173/quality
2. 切换到"偏差与CAPA"Tab
3. 查看CAPA跟踪表 (3条CAPA)
4. 进行中的CAPA点击"关闭验证" → 弹出签名确认

### 3.6 跨模块追溯
1. 打开 http://localhost:5173/trace
2. 点击"加载示例"
3. 确认7步追溯链完整展示 (委托→收样→分样→检测→ELN→复核→报告)
