# 客户自助门户设计文档

## 1. 产品调研

| 系统 | 客户门户功能 | HC-LIMS 差距 |
|------|-------------|-------------|
| **金现代 LIMS** | 在线委托、进度追踪、报告下载、合同查看、在线支付、消息推送 | 缺少支付、合同查看、消息中心 |
| **LabWare** | 客户 Portal：委托单管理、样品状态、报告在线查看、电子签名确认 | 缺少在线签名确认 |
| **StarLIMS** | 客户自助服务：订单管理、结果查询、趋势分析、合规报告 | 缺少趋势分析 |
| **赛默飞 SampleManager** | Web 客户门户：委托录入、样品追踪、报告查看、SOP 下载 | 基本对标 |

## 2. 功能架构

```
客户自助门户
├── 客户首页 (Home)      — KPI 看板 + 快速查询 + 通知
├── 我的委托 (Orders)    — 委托单列表 + 详情 + 进度追踪
├── 在线委托 (New Order)  — 检测项目选择 → 自动计费 → 提交
├── 报告下载 (Reports)    — 报告列表 + 在线预览 + PDF 下载
├── 合同管理 (Contracts)  — 合同查看 + 到期提醒
├── 账户信息 (Account)    — 公司信息 + 联系方式 + 信用等级
└── 消息中心 (Messages)   — 系统通知 + 客服消息
```

## 4. 组件树

```
CustomerPortalPage
├── Banner (渐变蓝色背景)
├── Tabs
│   ├── HomeTab
│   │   ├── KpiRow (5 cards: 委托/进行/报告/消费/待付)
│   │   ├── QuickSearch (委托单号搜索 + Steps 进度)
│   │   └── NotificationTimeline
│   ├── OrdersTab
│   │   └── OrdersTable (委托单列表 + 进度条)
│   ├── ReportsTab
│   │   └── ReportsTable (报告列表 + 预览/下载)
│   ├── NewOrderTab
│   │   ├── ContactForm (联系人/电话/样品/类型/地点)
│   │   ├── TestItemPicker (卡片式选择 + Tag.CheckableTag)
│   │   ├── PriceEstimate (实时计算)

│   └── AccountTab
│       ├── CompanyInfo (Descriptions)
│       └── ContactInfo

```

## 5. 样式规范

| 属性 | 值 |
|------|-----|
| 主色调 | `#1677ff` (蓝) |
| Banner 背景 | `linear-gradient(135deg, #1677ff, #0958d9)` |
| 卡片圆角 | `12px` |
| KPI 卡片 | 白色底、hover 阴影、灰色小图标 |
| 检测项目 Tag | `CheckableTag`、选中蓝色边框+浅蓝底 |
| 进度条 | 线形 Progress + Steps 垂直 |
| 弹窗 | 居中 Modal、浅灰背景 |
