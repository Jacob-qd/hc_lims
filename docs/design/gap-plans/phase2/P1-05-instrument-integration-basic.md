# P1-05 仪器数据直连（基础版）

> 对标: LabWare Instrument Integration (300+ drivers) | StarLIMS SDMS
> 版本: v1.0 | 优先级: P1 | 估算: 6周

---

## 1. 功能概述

实现仪器数据自动采集框架，从手动录入升级为自动采集。基础版覆盖文件解析和串口通信。

### 核心能力

- 通用数据文件解析（CSV/Excel/PDF）
- RS232 串口基础驱动框架
- 仪器连接配置管理
- 自动数据匹配与填充
- 采集异常告警

### 驱动路线图

```
基础版 (T+2~4月)          高级版 (T+4~8月)
├── CSV/Excel 文件解析      ├── HPLC (岛津/安捷伦/Waters)
├── RS232 串口框架          ├── GC (气相色谱)
├── 目录监听自动导入         ├── ICP-MS
├── Empower CSV 解析        ├── UV-Vis
└── Chromeleon CSV 解析     └── CDS API 集成
```

## 2. 功能需求

### FR-01: 文件解析

| 格式 | 解析方式 | 数据提取 |
|------|---------|---------|
| CSV/TSV | 按模板列映射 | 样品号、检测项、结果、单位 |
| Excel (.xlsx) | SheetJS 解析，按模板 | 同上 |
| PDF (结构化) | PDF text extraction | 同上 |
| ASTM E1394 文件 | 协议解析 | 完整的仪器输出数据 |

### FR-02: RS232 串口驱动

```
配置连接 → 打开串口 → 监听数据 →
按协议解析 → 标准化 → 存储原始数据 →
按配置文件映射 → 填充结果表
```

### FR-03: 目录监听

- 监听仪器输出目录（如 `C:\InstrumentData\Export\`）
- 新文件产生后自动触发导入流程
- 支持文件模式过滤（`*.csv`, `*_result.csv`）
- 导入后移动文件到 `processed/` 或 `failed/` 目录

│          Instrument Gateway Service               │
│                                                   │
│  ┌──────────┐ ┌──────────┐ ┌──────────────────┐  │
│  │ File     │ │ Serial   │ │ TCP/IP (未来)     │  │
│  │ Watcher  │ │ Driver   │ │ Driver           │  │
│  └────┬─────┘ └────┬─────┘ └────────┬─────────┘  │
│       │            │                │            │
│  ┌────┴────────────┴────────────────┴─────────┐  │
│  │           Plugin Interface                  │  │
│  │  · connect(config) → Connection             │  │
│  │  · read() → RawData                        │  │
│  │  · parse(rawData) → ParsedResult[]         │  │
│  │  · disconnect()                            │  │
│  └────────────────────┬───────────────────────┘  │
│                       │                          │
│  ┌────────────────────┴───────────────────────┐  │
│  │           Data Normalizer                   │  │
│  │  · 列映射 → 标准字段                        │  │
│  │  · 单位转换                                │  │

│  └────────────────────┬───────────────────────┘  │
│                       │                          │
