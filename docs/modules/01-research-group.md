## 1. 功能概述

高校科研场景的课题组管理，支持课题组的创建、成员管理、经费管理和成果汇总。

### 已实现功能

| 功能点 | 状态 | 说明 |
|--------|:----:|------|
| KPI 卡片（总课题数/在研/待结题/成果数） | ✅ | 4项指标 |
| 课题组列表（筛选+搜索） | ✅ | 按状态/负责人筛选 |
| 课题组详情（基本信息+成员+项目+经费+成果） | ✅ | Tab 页分组 |
| 成员管理（组长/组员/学生） | ✅ | 角色+列表 |
| 研究项目管理 | ✅ | 关联研究项目 |
| 经费管理（预算+支出+余额） | ✅ | 经费统计 |
| 成果汇总（论文/专利/奖励） | ✅ | 成果列表 |

## 2. 用户故事

### US-01: 课题组长管理课题组
> 作为**课题组长**，我创建课题组并管理成员、研究项目、经费和成果。

**验收标准：**
- [ ] 课题组列表: 名称/负责人/成员数/项目数/经费/状态
- [ ] 按状态筛选（在研/已结题/暂停）
- [ ] 详情 Tab: 基本信息/成员/项目/经费/成果
- [ ] 经费: 预算金额/已支出/余额 统计

### US-02: 研究生查看课题组成员
> 作为**研究生**，我在课题组页面查看成员名单和各自承担的研究任务。

## 3. API 接口

```
GET    /api/v1/research-groups              # 课题组列表
POST   /api/v1/research-groups              # 创建课题组
GET    /api/v1/research-groups/:id          # 详情
PUT    /api/v1/research-groups/:id          # 更新
POST   /api/v1/research-groups/:id/members  # 添加成员
DELETE /api/v1/research-groups/:id/members/:uid  # 移除成员
```
