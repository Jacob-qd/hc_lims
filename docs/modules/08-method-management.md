## 2. 用户故事

### US-01: 质量主管管理方法库
> 作为**质量主管**，维护实验室的检测方法库，确保所有方法有版本控制。

**验收标准：**
- [ ] 方法列表: 名称/编号/版本/类型/状态/部门/批准日期
- [ ] 按方法类型筛选（化学/物理/微生物/仪器）
- [ ] 状态标签颜色标识

### US-02: 检测员查看方法 SOP
> 作为**检测员**，执行检测前查看方法的标准操作程序，确保操作规范。

**验收标准：**
- [ ] 方法详情 Tab: 基本信息/验证数据/SOP/版本历史
- [ ] SOP Tab 展示操作程序和步骤
- [ ] 版本历史 Timeline 展示变更记录
- [ ] 支持查看历史版本

## 3. API 接口

```
GET    /api/v1/methods                      # 方法列表
POST   /api/v1/methods                      # 创建方法
GET    /api/v1/methods/:id                  # 方法详情
PUT    /api/v1/methods/:id                  # 更新方法
PUT    /api/v1/methods/:id/status           # 更新状态
GET    /api/v1/methods/:id/versions         # 版本历史
```
