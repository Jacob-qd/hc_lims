## 2. 用户故事

### US-01: 业务员管理客户信息
> 作为**业务员**，我在客户档案中维护客户信息，查看客户的委托历史和合同执行情况。

**验收标准：**
- [x] 客户列表: 名称/类型/联系人/电话/状态/合同数
- [x] 按客户类型（企业/政府/个人）筛选
- [x] 详情页: 基本信息 + 合同 Tab（显示关联合同列表）+ 委托单 Tab（显示关联委托单列表）+ 跟进记录 Tab（Timeline）
- [x] 详情页可操作：编辑、发送邮件
- [x] 客户新增/编辑/删除

### US-02: 合同管理员跟踪合同
> 作为**合同管理员**，我管理检测委托合同，跟踪合同的执行状态和回款情况。

**验收标准：**
- [x] 合同列表: 编号/客户/金额/签订日期/到期日期/状态
- [x] 状态标签: 执行中/即将到期/已到期/已终止
- [x] 合同详情 Tab: 基本信息/执行进度/回款记录
- [x] 合同终止: Popconfirm 确认后状态变为已终止
- [x] 合同续约: 即将到期/已到期合同可续约，状态恢复执行中
- [x] 回款记录: 在合同详情中添加回款（日期/金额/方式）
- [x] 合同新增/编辑/删除

### US-03: 商务人员管理委托单
> 作为**商务人员**，我录入和跟踪客户委托单，推进检测流程并记录付款情况。

**验收标准：**
- [x] 委托单列表: 编号/客户/项目/样品数/金额/状态/付款状态
- [x] 委托单详情 Tab: 基本信息/进度追踪/样品详情
- [x] 状态推进: 列表行和详情页均可点击推进，待处理→采样中→检测中→已完成
- [x] 标记付款: 一键标记全额付款，付款状态更新为已付款
- [x] 委托单新增/编辑/删除

### US-04: 商务人员管理报价单
> 作为**商务人员**，我创建报价单，选择检测项目和收费标准，发送给客户确认。

**验收标准：**
- [x] 报价单列表: 编号/客户/金额/有效期/状态
- [x] 状态流转: 草稿→已发送→已确认/已拒绝
- [x] 报价单详情: 展示检测项目清单（项目/方法/单价/数量/小计）
- [x] 生成委托单: 已确认报价可一键生成委托单
- [x] 报价单新增/编辑/删除
- [x] 新建报价时选择检测项目自动填充方法和单价

## 3. API 接口

```
GET    /api/v1/clients                      # 客户列表
POST   /api/v1/clients                      # 新增客户
PUT    /api/v1/clients/:id                  # 更新客户
DELETE /api/v1/clients/:id                  # 删除客户

GET    /api/v1/contracts                    # 合同列表
POST   /api/v1/contracts                    # 新增合同
PUT    /api/v1/contracts/:id                # 更新合同
DELETE /api/v1/contracts/:id                # 删除合同

GET    /api/v1/orders                       # 委托单列表
POST   /api/v1/orders                       # 新增委托单
PUT    /api/v1/orders/:id                   # 更新委托单
DELETE /api/v1/orders/:id                   # 删除委托单

GET    /api/v1/quotations                   # 报价单列表
POST   /api/v1/quotations                   # 新增报价单
PUT    /api/v1/quotations/:id               # 更新报价单
DELETE /api/v1/quotations/:id               # 删除报价单
```
