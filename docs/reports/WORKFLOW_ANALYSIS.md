# HC-LIMS 业务流程分析与补全计划

## 检测实验室版 完整流程闭环

```
客户委托/管理 ──→ 样品登记 ──→ 检测任务分配 ──→ 结果录入 ──→ 报告编制 ──→ 审核签发
    (clients)      (samples)      (tasks)        (result)     (reports)    (reports)
       │               │              │              │            │
       │               ▼              ▼              ▼            ▼
       │         质量控制 ◄──── 仪器管理 ◄──── 库存管理 ◄──── 方法管理
       │         (quality)    (instruments)  (inventory)    (methods)
       │
       └── 人员与培训 ──→ 系统管理 ──→ 数据分析
          (personnel)     (settings)    (statistics)
```

### 模块间联动需求

| 源模块 | 目标模块 | 联动方式 |
|--------|---------|---------|
| 样品管理 | 检测管理 | 样品行→查看关联任务 |
| 检测管理 | 样品管理 | 任务行→查看关联样品 |
| 检测管理 | 报告管理 | 任务完成→生成报告 |
| 报告管理 | 检测管理 | 报告行→查看关联任务 |
| 报告管理 | 样品管理 | 报告行→查看关联样品 |
| 质量控制 | 检测管理 | 偏差→查看关联任务 |
| 质量控制 | 仪器管理 | 失控→查看关联仪器 |
| 仪器管理 | 质量控制 | 校准逾期→创建偏差 |
| 库存管理 | 仪器管理 | 耗材→关联仪器 |
| Dashboard | 全部模块 | 快速操作+KPI点击跳转 |
| 仪器共享预约 | 课题组管理 | 预约→查看课题组 |
| ELN | 研究项目 | 记录→关联项目 |

## 当前缺失的功能闭环

### 1. 样品→任务 导航
- 样品详情页应显示关联的任务列表
- 点击任务编号可跳转到任务详情

### 2. 任务→报告 导航
- 任务完成后应有"生成报告"按钮
- 报告列表应显示关联的样品和任务

### 3. 质量→仪器/任务 导航
- 偏差详情中应链接到相关仪器
- 失控结果应链接到相关任务

### 4. 菜单联动
- Dashboard KPI点击跳转到对应模块（已部分实现）
- 侧边栏高亮当前模块（已实现）
- 面包屑导航（待实现）
