## 交互说明
- KPI卡片数字点击可下钻到对应列表页
- 最近样品行点击打开右侧详情抽屉
- 任务队列数字点击跳转对应模块
- 系统提醒点击跳转处理页面
- 快速操作点击跳转对应功能页
