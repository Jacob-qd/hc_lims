## 页面2：新建样品登记（5步向导）

### Step 1 — 基本信息
- 样品编号 *：自动生成 SMP+日期+序号（如 SMP202405210019），右侧显示"系统自动生成"
- 样品名称 *：输入框，placeholder"请输入样品名称，如：地表水样品-1"
- 样品类型 *：下拉选择
- 客户名称 *：下拉选择 + 右侧"+"新增客户按钮
- 来源项目：下拉选择
- 采样地点 *：输入框 + 地图定位图标
- 采样时间 *：日期时间选择器
- 送检人 *：下拉/输入 + 搜索图标
- 接收人 *：下拉/输入 + 搜索图标
- 接收日期：日期选择器（默认当天）
- 优先级：下拉（常规/高/紧急）
- 标签：下拉/输入多选
- 备注：文本框 0/200

**右侧实时预览面板**
- 样品概览：编号/名称/类型/客户/接收日期/状态（草稿）
- 条码预览：一维条码 + 编号文字
- 风险提示：✅ 常温保存要求已满足 / ⚠️ 部分检测项目设为加急 / ℹ️ 请尽快提交以安排检测
- 附件/采样照片：上传区（支持jpg/png/pdf/doc/xls，单文件≤20MB），缩略图预览

**底部按钮**：取消 | 保存草稿 | 下一步：采样信息 | 提交样品 | 打印条码

### Step 2 — 采样信息
- 采样人员、采样方式、采样条件、现场照片上传

### Step 3 — 容器与保存
多行表格：
| # | 容器类型 * | 规格 * | 数量 * | 保存条件 * | 操作 |
行操作：复制、删除
底部：+ 添加容器

### Step 4 — 检测项目
- 检测项目勾选列表（项目名称、方法、预计完成、分配实验室、优先级）
- 批量选择 / 清空选择
- 已选择 N 项显示
- 行操作：移除 / 添加
- 优先级列支持加急标记

⚠️ 提示：请至少选择1个检测项目

### Step 5 — 确认提交
- 汇总预览所有信息
- 条码预览
- 风险提示
- 附件列表
- 提交按钮

## 交互说明
- 表格行hover高亮
- 点击"详情"打开右侧抽屉
- 状态流转操作（接收/分配/开始检测等）需弹窗确认并填写理由
- 批量操作需先勾选行复选框
- 样品编号自动生成规则：SMP + yyyyMMdd + 3位序号
