   - 事件描述：2024-05-19 葡萄糖(GLU)水平1质控结果232.0 mg/dL，超出Westgard规则1₃ₛ，超出允许范围(90.0~110.0 mg/dL)。
   - 发现经过：日常质控结果自动审核发现异常，系统提示超出Westgard规则，质控负责人张伟复核确认。
   - 影响评估：可能影响2024-05-19该批次共86份患者样品的检测结果；已评估对临床决策影响为低风险，已启动回顾性评估。
   - 根因分析（5Why）：
     1. 为什么质控结果超出范围？→ 质控品测定值偏高。
     2. 为什么质控品测定值偏高？→ 试剂批号BC-6800可能出现性能漂移。
     3. 为什么试剂性能漂移？→ 试剂开瓶后未在规定时间内使用完毕。
     4. 为什么未在规定时间内使用完毕？→ 未严格执行试剂开瓶后28天有效期管理。
     5. 为什么未严格执行？→ 操作人员对试剂管理SOP理解不足，培训记录缺失。
   - 纠正措施：
     - ✅ 已隔离该批试剂BC-6800，停止使用；
     - 更换新批号试剂并重新检测质控，结果恢复在控。
     - 责任人：张伟 | 完成日期：2024-05-19 17:10
   - 预防措施：
     - 更新试剂开瓶后有效期管理SOP，明确标签与记录要求；
     - 加强试剂管理培训，覆盖相关人员。
     - 责任人：李娜 | 计划完成日期：2024-05-25
