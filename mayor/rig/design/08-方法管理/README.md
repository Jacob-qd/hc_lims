#### 章节1 — 适用范围
本方法规定了地表水中痕量六价铬的测定方法。本方法适用于地表水、地下水和生活污水中六价铬的测定，方法检出限为 0.01 μg/L。

#### 章节2 — 原理
在弱酸性条件下，六价铬与 1,5-二苯碳酰二肼形成紫红色络合物，于 540 nm 处测定吸光度，根据标准曲线定量。

#### 章节5 — 样品前处理
水样经 0.45 μm 滤膜过滤，取 100 mL 于锥形瓶中，调节样品 pH 为 3.0 ~ 4.0，待测。

#### 章节6 — 分析步骤
6.1 向锥形瓶中加入 2.0 mL 磷酸-乙酸缓冲溶液（pH 3.5）；
6.2 加入 2.0 mL 1,5-二苯碳酰二肼显色剂，摇匀；
6.3 于 25 ℃ 避光显色 10 min；
6.4 以试剂空白为参比，于 540 nm 波长处测定吸光度。
