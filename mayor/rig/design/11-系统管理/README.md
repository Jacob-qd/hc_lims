## 交互说明
- 权限矩阵支持批量勾选/取消
- 角色信息修改后需保存权限才生效
- 权限变更需审批流程（如果配置了）
- 数据范围支持自定义配置
- MFA开关影响该角色下所有用户登录方式
- 组织结构树支持拖拽调整部门层级
