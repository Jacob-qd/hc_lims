# 19-成果管理（高校场景）

## 设计图
| 文件名 | 页面 | 说明 |
|--------|------|------|
| 01-成果列表.png | 成果列表 | 搜索筛选+成果表格+类型标签 |
| 02-成果详情.png | 成果详情 | 成果元信息+关联实验数据+导出 |

## 页面1：成果列表
- 搜索：标题、作者、期刊
- 筛选：类型（论文/专利/报告）、年份、课题组、项目
- 表格：标题 / 类型 / 作者 / 期刊 / 年份 / DOI / 关联项目 / 操作
- 类型标签：论文(蓝)/专利(绿)/报告(橙)/数据(紫)

## 页面2：成果详情
- 论文元信息（标题/期刊/DOI/发表日期/作者列表）
- 关联实验数据列表
- 数据导出按钮
