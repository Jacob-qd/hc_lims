## 1. 功能概述

研究项目的全生命周期管理，从立项到结题，包含任务分配、进度跟踪和成果关联。

### 已实现功能

| 功能点 | 状态 | 说明 |
|--------|:----:|------|
| 项目列表（筛选+搜索） | ✅ | 按状态/类型/负责人筛选 |
| KPI 卡片（总项目/进行中/待结题/逾期） | ✅ | 4项指标 |
| 项目详情（基本信息+任务+进度+成果） | ✅ | Tab 页分组 |
| 任务分解 | ✅ | WBS 任务列表 |
| 进度跟踪（甘特图） | ✅ | 时间轴视图 |
| 经费管理 | ✅ | 预算/支出/余额 |
| 成果关联 | ✅ | 论文/专利/奖励 |
| 成员管理 | ✅ | 项目组成员 |

## 2. 用户故事

### US-01: 课题组长创建研究项目
> 作为**课题组长**，我在系统中创建研究项目，分解任务、分配负责人、跟踪进度。

**验收标准：**
- [ ] 项目列表: 名称/负责人/周期/经费/状态
- [ ] 详情 Tab: 基本信息/任务/进度/经费/成果/成员
- [ ] 任务以甘特图展示，可查看关键节点
- [ ] 经费: 预算/已使用/余额

## 3. API 接口

```
GET    /api/v1/research-projects            # 项目列表
POST   /api/v1/research-projects            # 创建项目
GET    /api/v1/research-projects/:id        # 详情
PUT    /api/v1/research-projects/:id        # 更新
GET    /api/v1/research-projects/:id/tasks  # 任务列表
POST   /api/v1/research-projects/:id/tasks  # 添加任务
```
