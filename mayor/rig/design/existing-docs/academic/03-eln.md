## 2. 用户故事

### US-01: 研究生撰写实验记录
> 作为**研究生**，我完成实验后在 ELN 中撰写实验记录，包括实验目的、方法、数据和结论。

**验收标准：**
- [ ] 创建实验记录: 填写标题、实验日期、实验目的、方法
- [ ] 富文本编辑器: 支持文字格式、图片插入、表格
- [ ] 支持上传附件（原始数据、图谱等）
- [ ] 保存草稿后可继续编辑

### US-02: 研究生锁定实验记录
> 作为**研究生**，实验记录完成后我进行签名锁定，锁定后不可再修改。

**验收标准：**
- [ ] 点击"签名锁定"后使用 SignaturePad 签名
- [ ] 签名后记录状态变为"已锁定"，编辑器变灰
- [ ] 锁定后所有修改操作被禁用
- [ ] 审计日志记录锁定的时间、操作人

### US-03: 导师见证实验记录
> 作为**导师**，研究生锁定实验记录后，我进行见证签名确认记录的真实性。

**验收标准：**
- [ ] 在待见证列表查看学生提交的实验记录
- [ ] 查看完整记录内容后点击"见证"
- [ ] 见证签名后状态变为"已见证"
- [ ] 见证后记录进入最终归档状态

## 3. API 接口

```
GET    /api/v1/eln-records                  # ELN 列表
POST   /api/v1/eln-records                  # 创建记录
GET    /api/v1/eln-records/:id              # 详情
PUT    /api/v1/eln-records/:id              # 更新草稿
POST   /api/v1/eln-records/:id/lock         # 签名锁定
POST   /api/v1/eln-records/:id/witness      # 见证签名
GET    /api/v1/eln-records/:id/versions     # 版本历史
```
