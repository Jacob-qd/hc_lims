## 1. 功能概述

大型仪器共享预约系统，支持仪器日历预约、时间冲突检测、使用记录追踪。

| 功能点 | 状态 | 说明 |
|--------|:----:|------|
| 仪器列表（可预约仪器） | ✅ | 展示可预约的仪器 |
| 预约日历（周视图） | ✅ | 按小时展示已占用/空闲时段 |
| 新建预约弹窗（选择仪器+时间+用途） | ✅ | 填写预约信息 |
| 时间冲突检测 | ✅ | 已有预约时段不可选 |

| 我的预约列表 | ✅ | 个人预约记录 |
| 使用记录 | ✅ | 预约完成后自动生成 |

## 2. 用户故事

### US-01: 研究生预约仪器
> 作为**研究生**，我需要预约使用液相色谱仪，在日历中选择空闲时段提交预约。

**验收标准：**
- [ ] 选择仪器后展示该仪器的预约日历（周视图）
- [ ] 已有预约时段灰色不可选
- [ ] 选择开始/结束时间、填写实验用途
- [ ] 提交后等待管理员审核

### US-02: 仪器管理员审批预约
> 作为**仪器管理员**，我在预约列表中审核用户的预约申请，确认仪器可用后通过。

**验收标准：**
- [ ] 待审核预约列表: 仪器/用户/时间/用途
- [ ] 点击"通过"或"驳回"操作
- [ ] 用户收到预约结果通知

## 3. API 接口

```
GET    /api/v1/reservations                 # 预约列表
POST   /api/v1/reservations                 # 创建预约
PUT    /api/v1/reservations/:id/approve     # 通过
PUT    /api/v1/reservations/:id/reject      # 驳回
PUT    /api/v1/reservations/:id/cancel      # 取消
GET    /api/v1/reservations/calendar?instrumentId=X&week=YY  # 日历数据
```
