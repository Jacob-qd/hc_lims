## 2. 用户故事

### US-01: 实验课老师创建课程
> 作为**实验课老师**，我在系统中创建实验课程，添加实验项目和学生名单。

**验收标准：**
- [ ] 创建课程: 课程名称、学期、班级、学时
- [ ] 添加实验项目: 实验名称、指导书、步骤、学时
- [ ] 导入学生名单（手动输入/批量导入）

### US-02: 学生查看实验安排
> 作为**学生**，查看本学期实验课程的安排，了解每次实验的内容和要求。

## 3. API 接口

```
GET    /api/v1/teaching/courses             # 课程列表
POST   /api/v1/teaching/courses             # 创建课程
GET    /api/v1/teaching/courses/:id         # 课程详情
PUT    /api/v1/teaching/courses/:id         # 更新课程
POST   /api/v1/teaching/courses/:id/students # 添加学生
POST   /api/v1/teaching/experiments         # 添加实验项目
PUT    /api/v1/teaching/experiments/:id/grade # 录入成绩
```
