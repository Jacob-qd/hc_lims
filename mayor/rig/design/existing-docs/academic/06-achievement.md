## 2. 用户故事

### US-01: 科研秘书登记论文
> 作为**科研秘书**，课题组成员发表论文后，在系统中登记论文信息。

**验收标准：**
- [ ] 选择类型为"论文"，填写标题、期刊、发表时间、作者
- [ ] 关联对应的课题组/研究项目
- [ ] 上传论文附件（PDF）
- [ ] 提交后在论文 Tab 中展示

### US-02: 查看课题组成果汇总
> 作为**课题组长**，按年度查看课题组的成果汇总，了解科研产出。

## 3. API 接口

```
GET    /api/v1/achievements                 # 成果列表
POST   /api/v1/achievements                 # 登记成果
GET    /api/v1/achievements/:id             # 详情
PUT    /api/v1/achievements/:id             # 更新
GET    /api/v1/achievements/export          # 导出
```
