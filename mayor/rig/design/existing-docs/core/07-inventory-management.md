## 2. 用户故事

### US-01: 库管员入库试剂
> 作为**库管员**，收到采购的试剂后，在系统中登记入库信息，更新库存数量。

**验收标准：**
- [ ] 点击"入库"弹出入库弹窗
- [ ] 选择或搜索品名，填写数量、批号、有效期、供应商
- [ ] 提交后库存数量自动更新
- [ ] 生成入库记录

### US-02: 检测员查看临期试剂
> 作为**检测员**，使用试剂前查看其有效期，确保不使用过期试剂。

**验收标准：**
- [ ] 列表页临期（30天内到期）行红色标记
- [ ] 已过期试剂灰色显示并标记"已过期"
- [ ] 排序: 按到期日降序

### US-03: 实验室主任发起采购申请
> 作为**实验室主任**，发现试剂库存不足时，在系统中发起采购申请。

**验收标准：**
- [ ] 在库存不足物品行点击"采购申请"
- [ ] 采购申请自动带入物品信息和建议数量
- [ ] 填写需求说明和期望到货日期
- [ ] 提交后进入审批流程

## 3. API 接口

```
GET    /api/v1/inventory                     # 库存列表
POST   /api/v1/inventory/inbound             # 入库
POST   /api/v1/inventory/outbound            # 出库
GET    /api/v1/inventory/alerts              # 预警列表
POST   /api/v1/inventory/purchase-request    # 采购申请
GET    /api/v1/inventory/purchase-requests   # 采购申请列表
```
