## 2. 用户故事

### US-01: 人事主管管理人员档案
> 作为**人事主管**，我维护实验室人员信息，包括部门、岗位、联系方式。

**验收标准：**
- [ ] 人员列表: 姓名/部门/岗位/手机/邮箱/状态
- [ ] 支持按部门/岗位筛选
- [ ] 新增/编辑人员档案弹窗

### US-02: 质量主管检查资质到期
> 作为**质量主管**，我查看人员资质到期情况，提前安排培训和复证。

**验收标准：**
- [ ] 资质台账: 姓名/资质名称/发证日期/有效期/状态
- [ ] 到期前 30 天黄色预警，过期红色标记
- [ ] 资质详情查看证书文件

## 3. API 接口

```
GET    /api/v1/personnel                    # 人员列表
POST   /api/v1/personnel                    # 新增人员
GET    /api/v1/personnel/:id                # 人员详情
PUT    /api/v1/personnel/:id                # 更新人员
GET    /api/v1/personnel/:id/certificates   # 资质列表
POST   /api/v1/personnel/:id/certificates   # 新增资质
```
