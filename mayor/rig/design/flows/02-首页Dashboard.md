# 首页 Dashboard — 用户使用流程

**设计图**: `design/01-首页/01-Dashboard首页.png`
**路由**: `/dashboard`
**状态**: ✅ 已实现 (303行)

---

## 主流程

```
[用户登录成功] → [/dashboard]
    → 加载 KPI 卡片
    → 加载最近样品表
    → 加载任务队列
    → 加载仪器状态
    → 加载趋势图/饼图
    → 用户可点击快速操作或查看详情
```

### 页面布局

```
┌─────────────────────────────────────────────────┐
│ KPI 卡片行 (4列)                                 │
│ 样品总数 | 待检测数 | 逾期任务 | 已完成报告      │
├──────────────────┬──────────────────────────────┤
│ 最近样品 (5条)    │ 任务队列                     │
├──────────────────┼──────────────────────────────┤
│ 仪器状态          │ 系统提醒                     │
├──────────────────┼──────────────────────────────┤
│ 周转时间趋势图     │ 样品类型分布饼图               │
├──────────────────┴──────────────────────────────┤
│ 快速操作行                                        │
└─────────────────────────────────────────────────┘
```

### 交互流程

| 动作 | 操作 | 预期 |
|------|------|------|
| 点击样品编号 | 跳转样品详情 | `/samples/:id` |
| 点击任务队列数字 | 跳转检测管理 | `/tasks` (filtered) |
| 点击快速操作"样品登记" | 跳转新建样品 | `/samples/new` |
| 点击快速操作"创建检测" | 跳转检测任务 | `/tasks/new` |
| 点击快速操作"查看报告" | 跳转报告管理 | `/reports` |
| 鼠标悬停图表 | 显示 tooltip | 数据详情 |
