# 样品管理 — 用户使用流程

**设计图**: `design/02-样品管理/01-样品登记列表.png`, `02-样品列表.png`, `03-新建样品登记-5步向导.png`
**路由**: `/samples`, `/samples/new`, `/samples/:id`
**状态**: ✅ 已实现 (457行)

---

## 主流程 A：样品登记

```
[样品管理 → 新建样品] → [5步向导开始]
    → Step 1: 基本信息 (编号自动生成/名称/类型/采样地点/时间/客户/优先级)
    → Step 2: 采样信息 (采样人员/方式/条件/现场照片)
    → Step 3: 容器与保存 (多行表格: 容器类型/规格/数量/保存条件)
    → Step 4: 检测项目 (勾选检测项目/方法/预计完成/分配实验室)
    → Step 5: 确认提交 (汇总预览/条码预览/风险提示/附件上传)
    → 提交 → POST /api/v1/samples → 跳转样品列表
```

## 主流程 B：样品浏览与筛选

```
[样品管理 → 样品列表] → [加载统计卡片/表格]
    → 使用筛选栏 (编号/客户/类型/日期/优先级/状态)
    → 点击行 → 右侧详情抽屉弹出
    → 点击「查看详情」→ 跳转样品详情页
```

## 主流程 C：样品流转

```
[样品详情] → [流转时间线]
    → 草稿 → 提交 → 待接收 → 接收 → 待分配 → 已分配 → 检测中 → 待审核 → 审核中 → 已完成
    → 每步操作: 接收/退回/分配/开始检测/提交审核/审核通过/退回
```

---

## 交互流程

| 动作 | 操作 | 预期 |
|------|------|------|
| 点击"新建样品" | 跳转登记向导 | Step 1 显示 |
| 向导 Step 1 → 2 | 填写基本信息后点"下一步" | 表单校验 |
| 向导 Step 4 选择检测项目 | 勾选项目复选框 | 右侧实时预览更新 |
| 向导 Step 5 提交 | 点击"提交" | POST → 跳转列表 |
| 点击表格行 | 展开右侧详情抽屉 | 动画滑入 |
| 点击状态标签 | 筛选该状态样品 | 表格刷新 |

---

## 状态机

```
[草稿] --提交--> [待接收]
[待接收] --接收--> [已接收] | --退回--> [待接收]
[已接收] --分配--> [待分配]
[待分配] --派工--> [已分配]
[已分配] --开始检测--> [检测中]
[检测中] --完成--> [待审核]
[待审核] --提交审核--> [审核中]
[审核中] --通过--> [已完成] | --退回--> [检测中]
[任意] --作废--> [已作废]
```
