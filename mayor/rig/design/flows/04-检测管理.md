## 主流程 A：任务总览

```
[检测管理] → [总览页]
    → KPI卡片: 待检测/检测中/待审核/已逾期
    → 我的任务看板 (按状态分组的任务卡片)
    → 检测任务列表 (完整表格)
    → 任务进度环形图
    → 检测流程步骤条
    → 仪器与任务排期甘特图
```

## 主流程 B：任务分配与排期

```
[总览 → 点击"分配"] → [任务分配页面]
    → 左: 待分配任务列表
    → 中: 排期视图 (甘特图/周视图/日历视图切换)
        → 横轴: 时间, 纵轴: 仪器/分析员
        → 拖拽任务卡片调整时间
        → 按仪器/按分析员切换
    → 右: 任务详情抽屉
    → 双击空闲时段 → 分配弹窗 (检测人员/时间/优先级/仪器/备注)
```

## 主流程 C：检测结果录入

```
[检测中任务 → 点击"录入结果"] → [结果录入页面]
    → 顶部: 样品概览栏 (编号/名称/项目/方法/送检人/日期)
    → 检测指标表格 (指标/结果/单位/检出限/限值/判定/备注)
    → 原始记录区 (文字描述)
    → 仪器读数区 (读数表格, 自动计算)
    → 计算过程区 (公式展示)
    → 附件上传 (原始记录/标准曲线/仪器读数/样品照片)
    → 审核信息区 (录入人/复核人/审核人)
    → 右: 样品概览 + 任务进度时间轴
    → 操作: 保存/提交复核
```

## 交互流程

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 总览查看KPI | 数字加载显示 |
| 2 | 点击"任务分配" | 进入排期视图 |
| 3 | 视图切换 (日/周/月) | 甘特图重绘 |
| 4 | 拖拽任务到时间轴 | 时间更新 |
| 5 | 双击空闲时段 | 分配弹窗 |
| 6 | 填写分配信息 + 确定 | POST → 任务分配 |
| 7 | 点击检测中任务 | 结果录入页 |
| 8 | 编辑指标表格 | 自动判定 |
| 9 | 输入仪器读数 | 自动计算结果 |
| 10 | 上传附件 | 文件列表更新 |
| 11 | 点击"提交复核" | POST → 状态→待审核 |

---

## 状态机

```
[待分配] --分配--> [待检测]
[待检测] --开始--> [检测中]
[检测中] --提交复核--> [待审核]
[待审核] --复核通过--> [已完成]
[待审核] --复核退回--> [检测中]
[任意] --逾期--> [已逾期] (自动)
```
