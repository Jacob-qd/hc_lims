# 质量控制 — 用户使用流程

**设计图**: `design/05-质量控制/01-偏差详情与CAPA处理.png`, `02-质量概览-LJ控制图.png`
**路由**: `/quality`, `/quality/deviations/:id`
**状态**: ⚠️ 待实现 (QualityPage.tsx 为 stub, 5行)

---

## 主流程 A：质量概览

```
[质量控制] → [质量概览]
    → KPI: 活动质控批次/失控事件/开放偏差/CAPA任务
    → L-J 控制图: 测定值/均值/±2SD/±3SD线
    → 质控样品结果表: QC批次/分析物/水平/靶值/测定值/偏差%/Westgard规则/状态
    → 非符合项/偏差列表
    → 审计就绪清单
```

## 主流程 B：偏差详情与CAPA

```
[点击偏差] → [偏差详情]
    → 偏差信息: 编号/来源/严重级别/状态
    → 处理流程进度: 发现→调查→审批→执行→验证→关闭
    → Tab: 偏差详情(5Why根因)/调查记录/CAPA流程/附件/质控趋势图
    → 审计追踪记录
```
