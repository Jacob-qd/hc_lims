## 主流程 A：仪器列表

```
[仪器管理] → [仪器列表]
    → KPI: 仪器总数/运行中/维护到期/校准逾期
    → 筛选: 名称/型号/编号/状态
    → 表格: 编号/名称/型号/厂家/位置/负责人/连接状态/校准日/维护日/利用率/状态
    → 状态分布环形图
    → 维护日历 (近30天)
    → 最近报警与事件
```

## 主流程 B：仪器详情

```
[点击仪器行] → [仪器详情页]
    → 顶部: 基本信息卡片 (图片/名称/型号/序列号/厂家/利用率/校准到期)
    → Tab: 基本信息/运行监控/校准记录/维护记录/使用预约/关联方法
    → 实时状态: 当前方法/样品/运行时长/流速/柱温/压力
    → 实时图表: 压力趋势图
    → 维护时间线
    → 校准证书列表
    → 预约日历 (周视图)
```

## 主流程 C：新建仪器

```
[点击"新建仪器"] → [新建仪器档案弹窗]
    → 字段: 名称/型号/编号/所属部门/购置日期/启用日期/状态/厂商/安装位置/责任人
    → 提交 → POST → 列表刷新
```
