# P1-01 条码批量打印

> 对标: StarLIMS Zebra/Sato 支持 | LabWare 条码打印
> 版本: v1.0 | 优先级: P1 | 估算: 1周

---

## 1. 功能概述

实现条码/二维码的批量生成与打印系统，支持多种标签模板和打印机配置。

### 核心功能

- 条码生成（Code128/Code39/EAN-13） + 二维码（QR Code）
- 标签模板设计器（尺寸/字体/条码位置/动态数据）
- 批量打印（多选→打印，支持 Zebra ZPL/PDF）
- 打印机配置与打印历史

### 标签类型

| 类型 | 用途 | 编码内容 |
|------|------|---------|
| 样品标签 | 样品瓶/袋标识 | 样品编号 + 类型 + 日期 + 二维码 |
| 试剂标签 | 试剂瓶标识 | 试剂名 + 批号 + 有效期 + 二维码 |
| 仪器标签 | 仪器资产标识 | 仪器编号 + 名称 + 二维码 |
| 位置标签 | 存储位置标识 | 位置编码 + 区域 + 二维码 |
| COC 标签 | COC 表单 | COC 编号 + 样品数 + 二维码 |

---

## 2. 功能需求

### FR-01: 标签模板

| 属性 | 说明 |
|------|------|
| 名称 | 如"样品标签-25x15mm" |
| 尺寸 | 宽度 × 高度 (mm) |
| 元素 | 文本、条码、二维码、矩形、线、图片 |
| 数据绑定 | `{{sampleNo}}`, `{{sampleType}}`, `{{date}}` |
| 字体 | 字号、字重、字体族 |
| 旋转 | 0°/90°/180°/270° |

## 3. 技术方案

```
前端                         后端
┌──────────────┐            ┌──────────────┐
│ jsbarcode    │            │ 打印配置 API  │
│ qrcode.react │            │ 打印日志      │
│ 模板渲染器    │            └──────────────┘
│ BrowserPrint │
│ → Zebra      │
│ → PDF下载    │
│ → 系统打印   │
└──────────────┘
```
