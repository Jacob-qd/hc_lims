# P1-06 多语言 (i18n)

> 对标: StarLIMS 20+ 语言 | LabWare 25+ 语言
> 版本: v1.0 | 优先级: P1 | 估算: 3周

---

## 1. 功能概述

为 HC-LIMS 增加国际化能力，支持多语言界面和区域化格式。第一阶段覆盖中英文。

### 核心能力

- React i18next 架构
- 按模块拆分翻译文件
- 语言切换（下拉/URL/持久化）
- 数据字典多语言化
- 区域化日期/数字/时区格式
- 翻译管理界面

---

## 2. 功能需求

### FR-01: 语言包结构

```
locales/
├── zh-CN/
│   ├── common.json        # 通用文字
│   ├── sample.json        # 样品管理
│   ├── test.json          # 检测管理
│   ├── report.json        # 报告管理
│   ├── quality.json       # 质量控制
│   ├── instrument.json    # 仪器管理
│   ├── inventory.json     # 库存管理
│   ├── personnel.json     # 人员管理
│   ├── system.json        # 系统管理
│   └── validation.json    # 校验提示
├── en/
│   └── ... (同上结构)
└── ja/ (未来)
    └── ...
```

### FR-02: 动态内容翻译

| 内容类型 | 翻译方式 | 存储 |
|---------|---------|------|
| UI 标签 | i18next key → value | JSON 文件 |
| 数据字典 | 数据库多语言列 | `label_zh, label_en` 字段 |
| 系统消息 | 模板变量 | 后端 i18n |
| 报告模板 | 模板可选语言 | 报告模板配置 |
| 错误提示 | i18next | JSON 文件 |

### FR-03: 区域化格式

| 格式项 | 中文 | 英文 |
|-------|------|------|
| 日期 | 2026-05-15 | 05/15/2026 |
| 时间 | 14:30 | 2:30 PM |
| 数字 | 1,234.56 | 1,234.56 |
| 货币 | ¥1,000 | $1,000 |
| 单位 | 毫米(mm) | inch(in) |
| 时区 | UTC+8 | UTC+0 |

## 3. 技术方案

```
┌──────────────────────────────────────────────┐
│           react-i18next                        │
│                                                │
│  LanguageDetector                               │
│  ├── localStorage                              │
│  ├── URL 参数 (?lang=en)                       │
│  └── 浏览器偏好 (navigator.language)           │
│                                                │
│  使用方式:                                      │
│  t('sample:create')  // "创建样品" / "Create Sample"
│  <Trans i18nKey="common:welcome">              │
│    欢迎, {{name}}                              │
│  </Trans>                                      │
│                                                │
│  数据字典:                                      │
│  <Select options={dict.map(d => ({              │
│    label: d[`label_${lang}`],                  │
│    value: d.value                               │
│  }))} />                                       │
└──────────────────────────────────────────────┘
```

### 工作流

```
1. 安装 react-i18next + i18next
2. 配置 i18n.ts (语言检测 + fallback)
3. 编写 codemod 提取所有硬编码中文 → JSON 翻译文件
4. 逐个模块替换: `"样品名称"` → `t('sample:name')`
5. 提供英文翻译
6. 添加语言切换 UI
7. 数据字典增加多语言列
```
