# P1-07 报告水印 + 防伪

> 对标: StarLIMS 动态水印 | 全部竞品二维码验真
> 版本: v1.0 | 优先级: P1 | 估算: 1周

---

## 1. 功能说明

动态水印 + 二维码防伪 + 摘要验真。报告生成/导出时自动添加，无需手动操作。

### 水印内容
```
"已签发" + 签发时间(YYYY-MM-DD HH:mm) + 签发人 + "报告编号: RPT-XXXX"
```

### 防伪机制
1. 报告 PDF 生成时计算 SM3 摘要，存入数据库
2. 报告页面右下角生成唯一验真二维码
3. 扫码 → 验真页面 → 查询签名记录 → 重新计算摘要 → 比对
4. 摘要一致 → ✅ 有效；不一致 → ❌ 已篡改

---

## 2. 技术方案

```
┌──────────────────────────────────┐
│  报告服务                          │
│                                    │
│  PDF 导出                          │
│  → @react-pdf/renderer 注入水印   │
│  → 水印层 = 半透明旋转文字        │
│                                    │
│  验真                              │
│  → 报告摘要 (SM3) → 存入 DB       │
│  → 二维码 (qrcode.react) → PDF    │
│  → 验真时前台计算摘要 → API 比对  │
└──────────────────────────────────┘
```
