# P2-05 移动 APP（原生）

> 对标: StarLIMS Mobile iOS/Android | LabWare Mobile
> 版本: v1.0 | 优先级: P2 | 估算: 12周

## 1. 功能概述

在 H5/PWA 基础上开发 Flutter 跨平台原生 APP，支持离线操作和完整的实验室移动办公场景。

### 核心功能

| 模块 | 功能 | 离线支持 |
|------|------|:--------:|
| 样品采集 | 拍照/GPS/语音/扫码 | ✅ |
| 样品签收 | 扫码签收+签名 | ✅ |
| 任务管理 | 待办列表+详情 | ✅ |
| 结果录入 | 手写/拍照/选择 | ✅ |
| 报告查看 | 预览/下载/分享 | ✅ |
| 签名确认 | 手写签名+密码 | ✅ |
| 消息推送 | 待办/超时/异常通知 | — |
| 数据同步 | 离线队列+自动同步 | ✅ |

### 离线策略

| 场景 | 策略 | 同步时机 |
|------|------|---------|
| 在线 | 直接 API 调用，同时缓存数据到本地 | 即时 |
| 离线 | 写入本地 SQLite + 操作队列 | 自动重连后同步 |
| 弱网 | 缓冲写入，批量提交 | 队列满或重连 |
| 冲突 | Last-Write-Win + 记录冲突日志 | 同步时解决 |
