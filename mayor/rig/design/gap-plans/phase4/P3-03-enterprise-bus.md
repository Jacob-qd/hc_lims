# P3-03 企业集成总线

> 对标: LabWare SAP Connector | StarLIMS ERP Integration
> 版本: v1.0 | 优先级: P3 | 估算: 12周

---

## 1. 功能概述

构建企业级集成总线，实现 HC-LIMS 与外部系统（ERP/CRM/MES/LIS）的数据交换和流程联动。

### 核心能力

| 集成场景 | 对接系统 | 数据流 |
|---------|---------|--------|
| 客户同步 | CRM / ERP | 客户信息 → LIMS |
| 订单同步 | ERP 销售模块 | 检测订单 → LIMS 样品 |
| 结果回写 | ERP 质检模块 | 检测结果 → ERP |
| 物料同步 | ERP 物料模块 | 试剂/耗材 → LIMS 库存 |
| 人员同步 | HR / AD | 员工信息 → LIMS 人员 |
| 设备同步 | EAM | 仪器信息 → LIMS 台账 |

### 集成模式

| 模式 | 说明 | 使用场景 |
|------|------|---------|
| API 网关 | 外部系统调用 LIMS API | 订单创建 |
| Webhook | LIMS 主动推送事件 | 结果回写 |
| 消息队列 | 异步可靠传输 | 大文件/批量同步 |
| 文件传输 | SFTP 文件交换 | 批量数据导入 |
| 数据库直连 | 只读视图 | 报表系统取数 |
