# 09-人员与培训 — 缺失设计图 Prompt

## 模块定位
人员与培训是 LIMS 的核心支撑模块，满足 CNAS-CL01 第 6.2 条"人员"要求，包括人员能力确认、授权、培训记录、资质证书管理。

## 需要你提供的设计图

### 页面1：人员档案列表
**请提供类似其他模块列表页的设计图，包含：**
- 顶部 KPI 卡片（总人数 / 在岗人数 / 资质到期预警 / 培训完成率）
- 高级筛选（姓名 / 部门 / 岗位 / 资质状态 / 入职日期）
- 人员表格（姓名 / 工号 / 部门 / 岗位 / 角色 / 所属实验室 / 入职日期 / 资质状态 / 操作）
- 右侧详情抽屉（基本信息 / 资质证书 / 培训记录 / 考核记录）

### 页面2：人员详情页
**请提供类似仪器详情页的设计图，包含：**
- 顶部个人信息卡片（头像 / 姓名 / 部门 / 岗位 / 在职状态）
- Tab 页导航：基本信息 / 资质证书 / 培训记录 / 考核记录 / 授权记录 / 审计追踪
- 资质证书列表（证书名称 / 发证机构 / 有效期 / 状态 / 附件）
- 培训记录时间线（培训课程 / 培训时间 / 考核结果 / 培训机构）
- 考核记录表格（考核类型 / 考核日期 / 结果 / 考官）

### 页面3：培训管理
**请提供类似库存管理列表页的设计图，包含：**
- 培训计划列表（计划名称 / 培训内容 / 培训对象 / 计划时间 / 状态）
- 培训实施记录（实际培训时间 / 讲师 / 参训人员 / 签到 / 考核成绩）
- 培训档案（按人员聚合展示所有培训记录）

### 页面4：资质管理
**请提供类似方法管理的设计图，包含：**
- 资质证书台账（证书编号 / 持证人 / 证书类型 / 发证机构 / 有效期 / 预警状态）
- 到期预警面板（30天内到期 / 60天内到期 / 已过期）
- 上岗证管理（岗位 / 授权项目 / 授权日期 / 有效期 / 签章）
- 能力确认记录（盲样考核 / 人员比对 / 留样再测 / 结果评价）

## 参考规范
- CNAS-CL01 6.2 人员要求
- RB/T 214-2017 4.2 人员要求
- 检验检测机构资质认定对人员的管理要求

## 设计风格要求
保持与已有模块一致：
- 深蓝色顶部导航栏（#1a3a5c 或类似）
- 左侧白色/浅色侧边栏
- Ant Design 5 表格风格
- 状态标签颜色：正常(绿) / 即将过期(黄) / 已过期(红) / 培训中(蓝)
- 右侧抽屉式详情面板
- 时间线组件展示培训/考核历史
