# 10-客户管理 — 缺失设计图 Prompt

## 模块定位
客户管理是第三方检测实验室的商务核心模块，覆盖客户信息、委托项目、合同管理、投诉处理、满意度跟踪，支持 CRM 基础功能。

## 需要你提供的设计图

### 页面1：客户列表
**请提供类似样品管理列表页的设计图，包含：**
- 顶部 KPI 卡片（客户总数 / 活跃客户 / 本月新增 / 即将到期合同）
- 高级筛选（客户名称 / 客户类型 / 行业 / 地区 / 合作状态 / 信用等级）
- 客户表格（客户编号 / 客户名称 / 客户类型 / 行业 / 联系人 / 联系电话 / 合作状态 / 累计样品数 / 操作）
- 右侧详情抽屉（基本信息 / 联系人 / 委托项目 / 历史样品 / 历史报告）

### 页面2：客户详情页
**请提供类似仪器详情页的设计图，包含：**
- 顶部客户信息卡片（Logo / 客户名称 / 客户类型 / 信用等级 / 合作状态）
- Tab 页导航：基本信息 / 联系人 / 委托项目 / 样品历史 / 报告历史 / 合同 / 投诉 / 满意度
- 委托项目列表（项目编号 / 项目名称 / 检测内容 / 合同金额 / 起止日期 / 状态）
- 样品历史表格（近期样品记录）
- 报告历史表格（近期报告记录）

### 页面3：委托项目管理
**请提供类似检测任务的设计图，包含：**
- 委托项目列表（项目编号 / 项目名称 / 客户 / 检测内容 / 合同金额 / 签约日期 / 截止日期 / 状态 / 进度）
- 项目详情（检测标准 / 检测频次 / 采样点位 / 合同附件）
- 项目进度跟踪（已完成的样品数 / 已出报告数 / 待处理数）

### 页面4：客户自助门户（可选）
**如有客户门户需求，请提供：**
- 客户登录后的首页（我的委托 / 样品进度 / 报告下载 / 在线委托）
- 样品进度查询（样品编号搜索 → 显示全流程状态）
- 报告下载列表（可下载 PDF / 在线预览）
- 在线委托申请（表单：样品信息 / 检测项目 / 联系人 / 备注）

## 参考规范
- 《检验检测机构资质认定管理办法》对客户管理的要求
- 第三方检测实验室 CRM 管理规范
- CNAS-CL01 7.1 要求、标书和合同评审

## 设计风格要求
保持与已有模块一致：
- 深蓝色顶部导航栏
- 左侧白色/浅色侧边栏
- Ant Design 5 表格风格
- 客户类型标签：企业(蓝) / 政府(绿) / 个人(灰)
- 合作状态：合作中(绿) / 暂停(黄) / 终止(红) / 潜在客户(蓝)
- 信用等级：A(绿) / B(黄) / C(红)
- 右侧抽屉式详情面板
