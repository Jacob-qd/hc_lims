# 13-课题组管理 — 设计图 Prompt

## 模块定位
课题组管理是高校科研实验室的核心组织单元，覆盖课题组设立、成员管理、经费预算、使用统计。

## 需要你提供的设计图

### 页面1：课题组列表
**请提供类似部门管理列表页的设计图，包含：**
- 顶部 KPI 卡片（课题组总数 / 在组总人数 / 可用经费余额 / 活跃项目数）
- 左侧组织树：「学院 → 系 → 课题组」，课题组显示名称 + PI + 成员数，点击切换
- 筛选栏（课题组名称、PI、所属院系、状态）
- 表格（课题组编号 / 名称 / 所属院系 / PI / 在组人数 / 经费余额 / 活跃项目 / 状态 / 操作）
- 新建课题组按钮 + 批量操作

### 页面2：课题组详情
**请提供类似仪器详情页的设计图，包含：**
- 顶部课题组信息卡片（Logo占位 / 名称 / PI / 所属院系 / 成立日期 / 状态）
- KPI 小卡片行（经费余额 / 在研项目数 / 在组人数 / 本月仪器使用次数）
- Tab 页导航：基本信息 / 成员管理 / 项目管理 / 经费台账 / 仪器使用统计
- 成员管理子页：成员表格（姓名 / 角色标签(颜色区分) / 学历 / 研究方向 / 加入日期 / 状态），添加成员按钮
- 角色标签颜色：PI(红) / 博士后(蓝) / 博士生(绿) / 硕士生(橙) / 科研助理(灰)

### 页面3：经费台账面板（作为详情页的子面板）
**请提供类似预算管理面板的设计图，包含：**
- 经费概览环形图（总经费/已使用/可用余额/预算执行率）
- 经费来源表格（项目编号 / 来源类型(纵向/横向) / 金额 / 已支出 / 余额 / 状态）
- 年度/项目切换筛选
- 导出Excel按钮

## 设计风格要求
保持与现有模块一致：
- Ant Design 5 表格风格
- 左侧组织树样式（参考 Ant Design Tree 组件）
- PI角色标签：红色描边
- 标签颜色区分角色
- 右侧详情面板 / 详情页完整布局
