# 14-研究项目管理 — 设计图 Prompt

## 模块定位
覆盖高校科研项目全生命周期（纵向/横向/校内），关联经费、实验、成果，满足科研管理需求。

## 需要你提供的设计图

### 页面1：研究项目列表
**请提供类似检测任务列表页的设计图，包含：**
- 顶部 KPI 卡片（在研项目数 / 经费总额 / 已使用经费 / 预算执行率）
- 高级筛选（项目编号、名称、类型(纵向/横向/校内)、PI、起止日期、状态）
- 项目类型标签：国自然(蓝) / 省基金(绿) / 教育部(紫) / 企业合作(橙) / 校内(灰)
- 状态标签：在研(绿) / 已结题(蓝) / 延期(黄) / 暂停(红)
- 表格（项目编号 / 名称 / 类型 / PI / 所属课题组 / 起止日期 / 经费总额 / 已用经费 / 执行率(进度条) / 状态 / 操作）
- 新建项目按钮

### 页面2：项目详情页
**请提供类似仪器详情页的设计图，包含：**
- 顶部项目信息卡片（名称 / 编号 / 类型 / 来源 / PI / 所属课题组 / 状态）
- 进度条（按时间进度 + 经费执行率双进度）
- Tab 页导航：
  - **基本信息**：项目简介摘要、成员列表、附件（任务书/合同）
  - **经费管理**（重点）：经费概览环形图、预算科目表(科目/预算/已支出/余额/支出占比)、支出明细表、超支预警(红)
  - **关联实验**：ELN记录列表（实验编号/名称/日期/实验人/Protocol）
  - **进度跟踪**：里程碑时间线
  - **成果关联**：论文/专利列表

## 设计风格要求
- 经费管理子页需要有预算科目表格 + 环形图的组合布局
- 超支科目标红提示
- 经费钻取：点击数字查看明细
- 时间线组件展示里程碑
