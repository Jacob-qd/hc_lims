# 15-电子实验记录ELN — 设计图 Prompt

## 模块定位
ELN（Electronic Lab Notebook）是科研实验室的核心工具，替代传统纸质实验记录本，支持富文本编辑、结构化记录、电子签名、导师审阅。

## 需要你提供的设计图

### 页面1：实验记录列表
**请提供类似文件管理或笔记列表的设计图，包含：**
- 顶部全文搜索栏（🔍 搜索标题、内容、标签、基因名、化学式）
- 筛选行（项目 / 课题组 / 实验人 / 日期范围 / 状态(草稿/已签名/已锁定) / Protocol类型）
- 视图切换：表格视图 / 卡片视图
- 表格列（实验编号(ELN-日期-序号) / 标题 / 实验人 / 所属项目 / 课题组 / 实验日期 / Protocol / 状态(草稿蓝/已签名绿/锁定灰) / 操作）
- 卡片视图：卡片展示标题 + 实验人 + 日期 + 状态标签 + 摘要预览
- 新建实验记录按钮

### 页面2：ELN编辑器（核心页面）
**请提供类似Notion/语雀笔记编辑器风格的设计图，包含：**

**顶部操作栏**：保存 | 提交签名 | 生成PDF | 关联项目 ▼ | 更多 ▼

**元信息区**（编辑器上方折叠/展开）：
- 实验编号（自动生成）
- 标题（大字输入框）
- 实验日期、实验人、所属项目（下拉选择）、关联Protocol（下拉选择）
- 标签（多标签输入）
- 实验目的（富文本，可简短）

**中间主编辑器**（占据大部分空间）：
- 富文本工具栏：H1-H4 / 加粗/斜体/下划线 / 列表 / 表格 / 插入图片 / 公式(LaTeX) / 代码块 / 引用 / 链接
- 编辑区：大面积空白编辑区域，支持文字、表格、图片混排
- 实验步骤（结构化，每步可标记完成状态）：1. 样品准备 [✅] / 2. 仪器设置 [⏳] / 3. 数据采集 [⏳]

**右侧信息面板**：
- 关联Protocol步骤预览
- 关联样品列表
- 关联仪器使用记录
- 附件列表（文件名称/大小/时间）
- 实验时间线（自动记录的操作历史）

**底部附件区**：拖拽上传区域 + 已上传文件列表（支持预览/下载）

### 页面3：签名与见证流程
**请提供类似审核弹窗设计图，包含：**

**实验人签名弹窗**：
- 提示文字：「确认实验记录完整，准备提交签名」
- 密码输入框（验证身份）
- 确认按钮
- 签名后显示：签名人、时间戳、IP地址

**导师审阅视图**（作为编辑器的一种模式）：
- 左侧：只读模式查看实验记录
- 右侧：批注面板（@人 / 回复 / 标记解决）
- 底部：签名区（密码验证 + 审阅意见 + 通过/退回按钮）

## 设计风格要求
- 参考 Notion / 语雀 / 飞书文档的编辑器风格，简洁清爽
- 左侧边栏 + 中间编辑区 + 右侧信息面板 三栏布局
- 工具栏浮在编辑区上方
- 公式用蓝色斜体显示（视觉区分）
- 签名弹窗需有安全感和正式感
- 锁定状态的记录显示🔒图标，不可编辑
