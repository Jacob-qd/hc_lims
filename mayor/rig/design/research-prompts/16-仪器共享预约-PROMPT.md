# 16-仪器共享预约 — 设计图 Prompt

## 模块定位
高校科研仪器共享预约系统，支持多课题组共享仪器、在线预约、自动计费、经费关联。

## 需要你提供的设计图

### 页面1：预约日历视图
**请提供类似日历/排期视图的设计图，包含：**

**顶部筛选**：
- 仪器选择器（下拉/搜索，显示仪器状态：🟢空闲 / 🔴运行中 / 🟡维护中 / ⚫故障）
- 视图切换：周视图 / 日视图 / 月视图
- 时间导航（< 本周 05-14 ~ 05-20 > | 今天）
- 按课题组筛选

**周视图日历（核心）**：
- 横轴：仪器名称（如 ICP-MS / 气相色谱 / 紫外分光光度计...）
- 纵轴：时间（08:00 ~ 22:00，30分钟一格）
- 预约块：彩色块（每课题组不同颜色），显示预约人简称 + 课题组
- 空闲时段：灰色/白色
- 维护时段：斜线填充 + 标注"维护中"
- 鼠标悬停预约块弹出详情卡片

**日视图**（切换后）：
- 单仪器全天的详细时间线
- 更精细的时间粒度（15分钟）

**新建预约弹窗**（点击空闲时段触发）：
- 仪器（已预填）
- 日期、时间段（已预选）
- 预约人（当前用户，自动填充）
- 关联项目/经费来源（下拉选择，显示余额）
- 用途说明（必填）
- 预估费用（自动计算显示）
- 确认预约按钮

### 页面2：计费规则设置
**请提供类似系统配置/规则编辑的设计图，包含：**
- 计费规则列表（每行一个仪器/规则组）
  - 仪器名称 / 计费方式(按时/按次/包时) / 常规费率 / 优先费率 / 免费时段 / 超时费率 / 状态
- 编辑弹窗：
  - 适用仪器多选
  - 计费方式下拉
  - 分级费率表格（用户类型 × 费率）
  - 免费时段设置
  - 超时/违规处罚设置
  - 生效日期

### 页面3：使用记录与统计
**请提供类似报表统计页的设计图，包含：**

**使用记录表格**：
- 预约编号 / 仪器 / 预约人 / 课题组 / 预约时间 / 实际上机时间 / 实际下机时间 / 使用时长 / 费用 / 状态
- 状态标签：已预约(蓝) / 使用中(绿) / 已完成(灰) / 未打卡(橙) / 已取消(红)

**统计图表区**：
- 本月仪器使用TOP10（横向柱状图）
- 各课题组仪器使用分布（饼图）
- 仪器利用率趋势（折线图，近30天）
- 违规统计卡片：超时次数 / 未打卡次数 / 取消次数

## 设计风格要求
- 日历视图参考 Google Calendar / Outlook 日历风格
- 预约块颜色按课题组区分（预置8-10种颜色）
- 每课题组颜色在图例中显示
- 计费规则表格参考 Ant Design 可编辑表格
- 操作按钮在每行右侧
