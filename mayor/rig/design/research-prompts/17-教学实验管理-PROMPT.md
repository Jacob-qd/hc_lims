# 17-教学实验管理（高校场景）— 设计图 Prompt

## 需要设计图

### 页面1：课程列表
**类似样品管理列表，包含：**
- KPI：本学期课程数、学生总人数、进行中实验、待批报告数
- 表格：课程编号 / 名称 / 授课教师 / 开课院系 / 学期 / 学生人数 / 实验数 / 状态(进行中/已结束)

### 页面2：课程详情
**Tab页**：
- **教学大纲**：实验目录列表（实验名称 / 学时 / 批次 / 状态）
- **学生管理**：学生名单表格（学号/姓名/分组/实验完成进度）
- **实验报告**：报告列表（学生/实验名称/提交时间/成绩/批阅状态）

## 设计风格
- 教学场景偏简洁风格
- 成绩输入支持批量编辑
- 状态标签：未提交(灰)/待批阅(橙)/已批阅(绿)
