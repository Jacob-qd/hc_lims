# 18-安全与废弃物管理（高校场景）— 设计图 Prompt

## 需要设计图

### 页面1：危化品台账
**类似库存管理列表，包含：**
- KPI：危化品总数、存量预警、过期预警、待处理废弃物
- 筛选：名称/CAS号/分类/存放位置/责任人
- 表格：名称/CAS号/危险分类/存量/存放位置/责任人/MSDS/状态(正常/预警/过期)
- 安全标签：剧毒(红)/易燃(橙)/腐蚀(蓝)/氧化(黄)

### 页面2：安全检查
- 日检/周检/月检切换
- 检查清单（逐项勾选：✅符合/❌不符合/—不适用）
- 隐患整改跟踪表

## 设计风格
- 安全模块风格偏严肃
- 危险等级用颜色区分
- 预警标红
